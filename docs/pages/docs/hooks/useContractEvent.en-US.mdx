# useContractEvent

Hook for subscribing to ethers Contract [events](https://docs.ethers.io/v5/api/contract/contract/#Contract--events).

```tsx
import { useContractEvent } from 'wagmi'
```

## Usage

The following examples use the [ENS Registry](https://etherscan.io/address/0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e#code) contract.

```tsx
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewOwner',
    listener: (event) => console.log(event),
  })
}
```

## Arguments

### addressOrName

Contract address or ENS name.

```tsx {5}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewResolver',
    listener: (event) => console.log(event),
  })
}
```

### contractInterface

Contract ABI in JSON or JS object format. An [ethers Interface](https://docs.ethers.io/v5/api/utils/abi/interface) is also allowed.

```tsx {6}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewResolver',
    listener: (event) => console.log(event),
  })
}
```

### eventName

Name of the event to listen to.

```tsx {7}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewResolver',
    listener: (event) => console.log(event),
  })
}
```

### listener

Callback that receives event.

```tsx {8}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewOwner',
    listener: (event) => console.log(event),
  })
}
```

### chainId (optional)

Force a specific chain id. The wagmi `Client`'s ethers `webSocketProvider` must be set up as a [chain-aware function](/docs/client#websocketprovider-optional) for this to work correctly.

```tsx {12}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewOwner',
    listener: (event) => console.log(event),
    chainId: 1,
  })
}
```

### once (optional)

Receive only a single event, then stop listener. Defaults to `false`.

```tsx {12}
import { useContractEvent } from 'wagmi'

function App() {
  useContractEvent({
    addressOrName: '0x00000000000C2E074eC69A0dFb2997BA6C7d2e1e',
    contractInterface: ensRegistryABI,
    eventName: 'NewOwner',
    listener: (event) => console.log(event),
    once: true,
  })
}
```
