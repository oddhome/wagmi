# useNetwork

Hook for accessing network data, such as current connected chain and [connector chains](https://wagmi.sh/docs/connectors/injected#chains-optional).

```tsx
import { useNetwork } from 'wagmi'
```

## Usage

```tsx
import { useNetwork } from 'wagmi'

function App() {
  const { chain, chains } = useNetwork()

  return (
    <>
      {chain && <div>Connected to {chain.name}</div>}
      {chains && (
        <div>Available chains: {chains.map((chain) => chain.name)}</div>
      )}
    </>
  )
}
```

> Tip: If you want the functionality to switch networks, [check out `useSwitchNetwork`](https://wagmi.sh/docs/hooks/useSwitchNetwork).

## Return Value

```tsx
{
  chain?: Chain & { unsupported?: boolean }
  chains: Chain[]
}
```
